# Documentation

ATSuite documentation is grouped by purpose:

- [ATG](./atg/): trace workflow, node configuration, and tool conversion notes.
- [User Guide](./userguide/): environment setup, build, deploy, and invocation guides.
- [Developer Guide](./developer-guide/): implementation notes for ATSuite provider integrations and internals.
- [Provider](./provider/): cloud provider and runtime background notes.

Most documents have paired English and Chinese versions:

- English: `name.md`
- Chinese: `name.zh.md`
